## G-Force_-_GPS_Data_Logger_for_iOS_Pythonista3
iOSアプリ「Pythonista 3」で動作する、高精度なGフォースメーター兼データロガーです。 iPhoneを車両に固定して走行することで、リアルタイムでのG（加速度）の可視化と、GPSを含めた詳細な走行データの記録が可能です。

## 概要
このスクリプトは、iPhoneの内蔵センサー（加速度センサーおよびGPS）を利用して、車両の挙動を計測します。

モータースポーツ、走行会、または安全運転の分析などに活用できます。

## 主な機能
リアルタイムGメーター: 3軸（前後・左右・上下）の加速度を合成し、視覚的に表示。

ピークホールド: 走行中の最大Gを記録し、画面上に保持。

データロギング: 加速度、位置情報（緯度・経度）、算出速度をCSV形式で保存。

速度算出: GPS座標の差分から時速(km/h)をリアルタイムに計算。

警告表示: 1.1Gを超えると背景が赤く変化し、限界域での走行を直感的に通知。

## 設置方法
推奨環境: iPhoneを車載スマホホルダーなどで**垂直（縦向き）**に固定してください。

向き: 画面が運転手の正面（目と平行）を向くように設置することで、前後左右のGが正しくプロットされます。


## 使い方
Pythonista 3でスクリプトを実行します。

実行と同時にGフォースの可視化が開始されます。

画面下の [START REC] ボタンをタップすると記録が始まります。

走行終了後、 [STOP REC] をタップすると、iPhone内の ../data フォルダにCSVファイルが保存されます。

画面の任意の位置（ボタン以外）をタップすると、最大G（MAX G）がリセットされます。


## 記録されるデータ項目（CSV）
保存されるCSVファイルには以下のデータが含まれます：

項目	説明

Date	記録年月日 (YYYY-MM-DD)

Time	記録時刻 (ミリ秒まで)

G_X	左右の加速度 (Lateral G)

G_Y	上下の加速度 (Vertical G)

G_Z	前後の加速度 (Longitudinal G)

G_Combined	3軸合成加速度 (Total G)

Lat / Lon	GPSによる現在地の緯度・経度

Speed_kmh	座標差分から計算された現在の時速
